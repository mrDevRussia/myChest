# JET IDE Installation Guide

## Installation Options

### Option 1: Using the All-in-One Executable Installer (Recommended)

1. Locate the `JET-IDE-Installer.exe` file in the project directory
2. Double-click the file to run the installer
3. The installer will extract all application files, install them to your local application data folder, create shortcuts, and launch the application automatically

### Option 2: Using the Simple Installer Batch File

1. Locate the `simple-installer.bat` file in the project directory
2. Double-click the file to run the installer
3. The installer will copy all necessary files to your desktop and create a shortcut

### Option 3: Manual Installation

1. Create a directory where you want to install JET IDE
2. Copy the entire `JET.App` directory to your installation directory
3. Copy the `LICENSE`, `README.md`, and other documentation files
4. Create a shortcut to `JET.App\JET.exe` on your desktop

## System Requirements

- Windows 10 or later
- .NET 9.0 or later
- 4GB RAM (minimum)
- 100MB free disk space

## Troubleshooting

If you encounter any issues during installation:

1. Ensure you have the .NET SDK installed
2. Try running the installer as administrator
3. Check that you have sufficient disk space
4. Verify that you have the necessary permissions to install software

## Uninstallation

To uninstall JET IDE:

1. Delete the JET IDE directory from your installation location
2. Delete the desktop shortcut
